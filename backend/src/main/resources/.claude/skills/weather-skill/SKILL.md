# 天气查询技能

## 技能简介
本技能通过调用 wttr.in 天气 API，获取指定城市的实时天气信息。支持全球大部分城市，返回格式化的中文天气报告。

## 指令说明
当用户询问天气时，请按以下步骤操作：

### 1. 确认查询参数
- 从用户查询中提取城市名称
- 如果用户未明确指定城市，应主动询问具体城市名称
- 城市名称支持中文、英文或拼音格式

### 2. 调用天气查询脚本
使用以下命令调用技能附带的 Python 脚本：
```bash
python scripts/weather.py "城市名称"
```

支持查询多个城市（用空格分隔）：
```bash
python scripts/weather.py "北京" "上海" "广州"
```


### 3. 处理返回结果
脚本将返回格式化的天气信息，包括：
- 城市名称
- 当前天气状况描述
- 温度（摄氏度）
- 体感温度
- 湿度百分比
- 风力方向和速度
- 观测时间
- 穿衣建议

### 4. 返回用户友好的响应
将脚本返回的格式化为用户提供清晰的天气报告，可适当添加表情符号增强可读性。

## 使用场景示例

**用户查询**: "今天北京天气怎么样？"
**技能调用**: `python scripts/weather.py "北京"`

**用户查询**: "北京、上海、广州的天气"
**技能调用**: `python scripts/weather.py "北京" "上海" "广州"`

**用户查询**: "What's the weather like in New York?"
**技能调用**: `python scripts/weather.py "New York"`

## 注意事项
1. 本技能依赖网络连接访问 wttr.in API
2. 城市名称区分大小写，建议使用标准城市名
3. 查询频率过高可能导致 API 限制
4. 脚本使用 Python 标准库，无需额外安装依赖包
5. 如遇网络问题，可建议用户稍后重试
